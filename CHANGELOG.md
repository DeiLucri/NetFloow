# Changelog — NetFloow

## [1.0.0] — 2026-05-09

### Initial release

- Theme core `netfloow.css` — 15 sections documentées
- Variables CSS complètes en format RGB dans `:root`
- Fonts Google : Bebas Neue (logo) + Outfit (UI)
- Header transparent avec logo NetFloow en Bebas Neue rouge
- Sidebar minimaliste avec accent rouge au survol
- Cards avec hover scale + overlay dégradé
- Barre de progression rouge (cartes + player)
- Boutons rouge avec hover assombri
- Support plugin Media Bar (IAmParadox)
- Support plugin Home Screen Sections (IAmParadox)
- Compatibilité KefinTweaks et Jellyfin Enhanced
- Page login personnalisée avec logo + tagline
- Episode previews stylisés
- Cast & Crew stylisés (cartes circulaires)
- Responsive mobile (max 768px)
- Responsive TV (min 1920px)
- Variant OLED : `variants/netfloow-oled.css`
- Variant Bleu : `variants/netfloow-blue.css`
- Addon scrollers : `addons/scrollers.css`
- Addon Media Bar : `addons/mediabar.css`
- Addon login : `addons/login.css`
- Logo SVG vectoriel : `assets/logo.svg`
